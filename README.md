## How to use it

```sh
$ git clone https://github.com/adam17/dataStructures
$ touch index.js
$ echo "'use strict'
const dataStructures = require('./dataStructures/')
const Queue = dataStructures.Queue
let q = new Queue()
q.enqueue(1)
console.log(q)" >> index.js
$ node index.js
```
